Class Queue<Type>

Queue typed similar to a std::queue with possibility to limit the capacity like a FIFO

Example

const queue = new Queue<number>(2);
queue.push(1); // [1]
queue.push(2); // [1,2]
queue.push(3); // [2,3] 1 has been removed to respect 2 capacity
